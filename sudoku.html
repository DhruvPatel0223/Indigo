How to play Sudoku

Goal

The goal is to fill in all the empty cells on the board with the correct numbers.

Rules

Cells each take a number between 1 and 9. A number may only appear once in a single line, row, column, or three-by-three box.

How to play

Click a cell to select it (in yellow). Then type a number to set the number in the selected cell. A cell will turn red if a wrong number is put into the cell.
